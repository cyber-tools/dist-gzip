<h2 align="center">Introduction</h2>
<div align="center">
  根据zip.config.js文件压缩成gzip
</div>
<h2 align="center">Install</h2>

```bash
npm install @tianque/dist-gzip -g
```

<h2 align="center">Usage</h2>

创建压缩配置文件zip.config.js:
```bash
dist-gzip config
```

根据zip.config.js执行压缩
```bash
dist-zip start
```
